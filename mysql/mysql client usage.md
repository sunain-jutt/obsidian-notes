
| connection | mysql -h *hostname or ip*  -u *username*  -p *password* |
| ---------- | ------------------------------------------------------- |

**root** (username) does not require password to login mysql.

## commands 
to view all the databases 
```
show database
```

to select a database
```
use [database name]
```

to list tables in the database
```
show [table name]
```

to see the whole content of the table
```
select * from [table name]
```
